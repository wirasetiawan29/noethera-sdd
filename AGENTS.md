# Agent context

Load these paths as context for AI coding agents:

- `.agent/workflows/` — SDD workflow and phase checklists
- `.agent/rules/` — Guardrails (always_on + model_decision)
- `.agent/skills/` — On-demand how-to (debugging, frontend, code review, etc.)

See `.agent/MANIFEST.yaml` for full index. Run `make validate` to check structure.
